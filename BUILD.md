# Build Scripts for Bertsheim

## Build All
To build everything in the workspace, run:

```bash
pnpm run build
```

## Build Windows Installer
To build the Windows installer, run:

```bash
pnpm run build:windows
```

## Run Tests
To run the tests, execute:

```bash
pnpm test
```

## Start Development
To start the development server, run:

```bash
pnpm start
```
